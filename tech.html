toronto urban form

generating various measures of urban form, built environment, and walkability for the Toronto and surrounding municipalities

canada transit access

code for computing, analyzing, and visualizing measures of transit accessibility for Canadian cities

linking Canadian historical census tract data to a common set of geographic boundaries via areal interpolation procedures

figure-ground population density

interpolating population data to create a more refined map of population density

equal-area lon-lat buffers

computes equal-area circular buffers for an input lon/lat coordinate in JavaScript

conflation algorithm for (topological) vector data

scripts designed to fix the spatial mismatch in 1991 and 1996 Canadian census boundaries and representative coordinate points

nearby transit trip frequency

interactive web-map of postsecondary student travel patterns

toronto address geocdoer

a custom geocoder that computes points from address strings for Toronto built for quick geocoding of large datasets (e.g. over 10,000)

collection of open-source .sh and .py scripts for wrangling spatial data
